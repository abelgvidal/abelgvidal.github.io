Mucho Ruido y Cuatro Plataformas

Posted: 2025-12-15   |   Tags: architect, platform

Hay cuatro tipos de plataformas: Marketplaces: son sitios que permiten a dos partes hacer transacciones. Ejemplos reales: Wallapop eBay Airbnb Plataformas base: dotan de recursos IT para empezar un proyecto. Ejemplos reales: Amazon Web Services (AWS) Microsoft Azure Google Cloud Platform Plataformas de desarrollo: permiten desarrollar mejor y más rápido. Ejemplos reales: GitHub Heroku Vercel Plataformas de capacidades de negocio: crean un ecosistema abierto. Ejemplos reales: Salesforce AppExchange Shopify App Store Apple App Store También puede haber modelos híbridos que combinan más de un tipo de plataforma. Read more...
